Biomes Plguin for Unity

I developed a **Biomes Plugin** for Unity that enhances terrain generation by auto-populating instanced objects based on a terrain splatmap. The plugin uses raycasting to determine where objects should be placed on the terrain, ensuring a seamless integration with the environment.

The plugin is customizable, allowing users to filter placement based on specific data for each terrain. This includes settings like the types of items that can be placed, their density, randomness, and slope limits. Additionally, it allows for density variations based on height, including height cutoffs to optimize placement.
